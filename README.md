# Design Shorts

Design Shorts is a zero-cost MVP for mobile-first HLD prep. It delivers one short topic per screen, remembers progress locally, and keeps a separate tech news digest backed by a free GitHub Actions pipeline.

## MVP features

- Vertical swipe HLD feed
- `Serial` mode with checkpoint resume
- `Random` mode with `75% unread / 25% read` weighting
- Explicit `Mark as Read` progress tracking
- Separate tech news feed
- Local-first storage with offline HLD content

## Stack

- Expo React Native
- TypeScript
- AsyncStorage for local checkpoints and settings
- Static `public/news.json` digest generated by GitHub Actions

## Run locally

```bash
npm install
npm start
```

## News digest pipeline

The app fetches a static JSON digest from a hosted URL and falls back to bundled or cached data if that fetch fails.

1. Set `EXPO_PUBLIC_NEWS_DIGEST_URL` to your GitHub raw JSON URL.
2. Push this repo to GitHub.
3. Enable the `Refresh News Digest` workflow.
4. The workflow writes `public/news.json` on a schedule.

You can also generate the file locally:

```bash
npm run generate:news
```
